# DARK MODE ISSUE ANALYSIS REPORT - Alunex Pulse App

## EXECUTIVE SUMMARY

The Alunex Pulse App is a React-based project management application. It implements a CSS variable-based dark mode system, but has CRITICAL HARDCODED COLORS preventing proper theme switching.

## 1. FRAMEWORK & TECH STACK

Framework: React 19.1.1 with Vite 7.1.7
Theme System: CSS Variables with data-theme attribute
Storage: localStorage with key "theme" (values: "light" or "dark")

Key Files:
- App.jsx: Initializes theme
- Settings.jsx: Contains theme toggle
- index.css: Master CSS variables

## 2. HOW DARK MODE WORKS (CORRECTLY)

The app sets data-theme="dark" on <html> when dark mode is selected.
CSS variables automatically change based on this attribute:

Light Mode (:root):
  --bg-main: #f8fafc
  --bg-white: #ffffff
  --text-dark: #1e293b

Dark Mode ([data-theme="dark"]):
  --bg-main: #0a0e1a
  --bg-white: #151b2b
  --text-dark: #e2e8f0
